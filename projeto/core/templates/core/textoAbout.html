Projeto Final de Graduação no Curso Sistemas de Informação da Universidade Franciscana, do aluno Marcelo Silva de Siqueira, sob orientação do professor Alexandre Zamberlan.
Este trabalho visa complementar o sistema projetado e implementado por Aloísio Kneipp dos Santos, nominado de "Sistemas Pervasivos Integrados por Agentes Inteligentes em JASON e Raspberry Pi", via o projeto e a implementação de um sistema Web para gestão de usuários e seus perfis em ambientes, como cômodos em uma casa, no que se refere a preferências de iluminação e temperatura.

O sistema Web integra dados de preferências e de usuários na simulação criada por meio de um Web Service. Para isso, as tecnologias Python, Django e Bootstratp são utilizadas para a construção dessa plataforma.
